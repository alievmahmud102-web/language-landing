# Sales reply — prompt template

Use to draft **short replies** to prospects (email or messenger). Format: **solution → explanation → next steps**.

## Inputs

- Their last message (paste verbatim).
- Your goal: book call, send deck, qualify, handle delay.
- Constraints: price floor, timeline, languages (RU client → Russian message body).

## Output

1. **Draft reply** (concise; optional 2 variants: soft / direct).
2. **Why** this structure (1–3 bullets).
3. **Next steps** for you (e.g. send calendar link, log in CRM).

## Example pattern

> Thanks for the detail on [X]. We typically help centers turn inquiries into enrollments with [one-line value].  
> Suggest a 20-min call [day options]: [link].  
> If easier, reply with your current site + how you track leads today.

## Guardrails

- No false promises; if unsure, propose discovery.
