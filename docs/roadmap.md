# Roadmap

Priorities as **Now / Next / Later**. Revisit after each client cycle or monthly planning.

## Now

- Harden **lead capture** end-to-end: forms → API → CRM list → no duplicate chaos (rules TBD).
- **Mobile** polish on landing and core CRM screens used daily.
- **Auth** and basic **role** separation (owner vs staff) if not already complete.
- **Dashboard MVP**: new leads, simple funnel or stage counts, source breakdown.
- **Telegram** path reliable for the primary use case (notify or handoff—match implementation).
- Materials to support **first repeatable sales**: one-pager, demo script, commercial outline (`docs/marketing/`).

## Next

- **3–5 case studies** with before/after metrics (response time, conversion, source clarity).
- Deeper **analytics** (cohorts, SLA to first contact) if data exists.
- **Onboarding checklist** for new centers (content, DNS, tracking).
- Export (CSV) for leads if customers ask; rate limits and abuse protection on public endpoints.
- Optional **NestJS** migration or module extraction only if complexity forces it—avoid premature split.

## Later

- Multi-branch or franchise org models if demand appears.
- Richer integrations (calendar booking, ad platforms, external CRM sync).
- White-label packaging or partner channel—only with clear economics.

## Goals reminder (3–6 months)

- Stable deal flow in target **$700–$2,000** band (update with real packages).
- Product perceived as a **system**, not a one-off site build.
