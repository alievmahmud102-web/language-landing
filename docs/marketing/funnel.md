# Funnel

## Top of funnel — awareness

- Channels: local SEO, Instagram / VK / Telegram communities, referrals, partnerships with schools, paid search/social (when budget exists).
- Asset: **landing** optimized for **one primary CTA** (call, form, messenger).

**Metric examples**: sessions, scroll depth (if tracked), CTA clicks.

## Middle — consideration

- Fast reply (human); optional FAQ on landing; short voice note or template messages—**process**, partly outside software.
- CRM ensures **no lead disappears**.

**Metric examples**: form start → submit rate, % leads with assigned owner within X hours.

## Bottom — decision

- Trial lesson / diagnostic / enrollment conversation.
- CRM stages reflect reality: `trial_scheduled` → `trial_done` → `won` / `lost`.

**Metric examples**: trial show rate, trial → paid conversion, average days to win.

## Retention / expansion

- Happy centers refer; second location or new program page.
- Track **referral source** on new leads.

## Attribution hygiene

- Use **UTM** on every paid link; consistent naming convention (`utm_source`, `utm_medium`, `utm_campaign`).
- For offline: manual **source tag** on lead (e.g. “walk-in flyer”).

## Dashboard alignment

- Dashboard should mirror this funnel so owners trust the numbers—define **stage definitions** once and reuse in sales conversations.
