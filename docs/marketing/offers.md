# Offers

> Replace placeholder numbers with **signed** pricing. Band discussed: **~$700–$2,000** per engagement depending on scope.

## Core offer (system)

**“Enrollment pipeline in a box”** — not a standalone website.

Includes (typical):

1. **Conversion landing** (structure + implementation; client provides text approval).
2. **CRM** for leads and basic student record after win.
3. **Dashboard** for inquiries, stages, and sources.
4. **Integrations**: forms + Telegram (as scoped).
5. **Simple admin** for day-to-day operations.
6. **Handover**: short training + doc checklist.

## Packaging ideas (illustrative)

| Tier | Scope | Notes |
|------|--------|--------|
| **Launch** | Single center, one landing, CRM+Telegram+forms, dashboard MVP | Fastest time-to-value |
| **Growth** | + extra landing variants / second location page / export | When they run ads at scale |
| **Partner** | Custom integration or deeper CRM fields | Priced after discovery |

## Add-ons (optional)

- Copywriting refresh (Russian) if client has only bullet points.
- Photography / video not included unless subcontracted.
- Ongoing retainer (small monthly): monitoring, tweaks, analytics review.

## Payment terms (template)

- **Deposit** to start (e.g. 40–50%) + milestone or balance on go-live—set what protects your cash flow.
- Clear **change request** rule: out-of-scope = separate estimate.

## Guarantees

- Avoid promising specific enrollment numbers; prefer **delivery**, **response process**, and **measurement** guarantees you can meet.
